# nrf5-temperature-beacon
 a simple BLE beacon advertising temperature and pressure from BMP-280 module.
 Use with https://github.com/gogela/nrf5-bme280-twi driver
 
 requires nRF SDK nRF5_SDK_17.1.0_ddde560
